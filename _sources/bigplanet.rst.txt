BigPlanet
-----
A suite of data analysis and extraction routines for ``VPLanet``. In many cases,
VPLanet is used to explore parameter space, a process that can generate large
amounts of data. BigPlanet is a python script for storing and accessing those
data via the HDF5 file format.

Installation
============
In the bigplanet directory in the VPLanet repository, run the setup script:

.. code-block:: bash

    python setup.py install


Usage
========

An `iPython notebook <https://github.com/VirtualPlanetaryLaboratory/vplanet-private/blob/master/bigplanet/examples/data_example.ipynb>`_
provides an in-depth tutorial.
